Voice Assistant Setup

Add transactions using Siri or Google Assistant

iOS - Siri Shortcuts

1
Open Shortcuts App

Open the Shortcuts app on your iPhone

2
Create New Shortcut

Tap + → Add Action → Web → Open URLs

3
Add URL Template
https://kaleb-ez-tracker.netlify.app/?action=add&amount=500&category=Food&type=expense¬e=Lunch

Replace amount and category with Ask Each Time variables

4
Add Variables

Tap on "Ask Each Time" for amount and category

URL Template:
https://kaleb-ez-tracker.netlify.app/?action=add
&amount=Ask Each Time
&category=Ask Each Time
&type=expense
¬e=Siri Added
5
Add to Siri

Tap Add to Siri → Record phrase like "Add expense to Kaleb Tracker"

Android - Google Assistant

1
Open Google Assistant Settings

Say "Hey Google, open Assistant settings"

2
Create Routine

Go to Routines → New Routine

3
Add Trigger Phrase

Example: "Add expense to Kaleb"

4
Add Action

Add "Open URL" action with this template:

https://kaleb-ez-tracker.netlify.app/?action=add&amount=$amount&category=$category&type=expense¬e=Assistant
